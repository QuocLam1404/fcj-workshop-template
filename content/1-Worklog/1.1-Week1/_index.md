# WEEK 1 WORKLOG

### Week 1 Objectives:

* Register and secure a new AWS account using best practices.
* Setup IAM permissions and multi-factor authentication (MFA) to protect the account.
* Establish budget thresholds and billing alerts to monitor Free Tier utilization.
* Deploy initial core services (EC2/S3) to get comfortable with the console navigation.
* Review completed tasks against the program's requirements for certification.

### Week 1 Achievements:

* **Successful Account Initialization**: Successfully registered a personal AWS account and passed verification steps.
* **Secured AWS Environment**: Followed the principle of least privilege by creating a separate IAM User for daily actions and activated MFA.
